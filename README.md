# Restaurant Landing Page Template

A responsive restaurant landing page template built with Tailwind CSS and vanilla JavaScript. The project renders a polished restaurant website experience with hero content, about section, services, testimonials, team profiles, reservation form, FAQ, and footer navigation.

## Features

- Clean restaurant landing page design
- Responsive navigation with mobile sidebar menu
- Tailwind CSS via CDN (`@tailwindcss/browser`)
- Hero section with call-to-action button
- About Us and service cards
- Customer reviews/testimonials section
- Team member showcase
- Reservation form with contact details
- FAQ section and footer links
- Image and icon assets included in `img/`

## Project Structure

- `index.html` — main landing page markup
- `img/` — image assets used by the template
  - `avatars/`
  - `favicon/`
  - `Home/`
  - `icons/`

## Usage

1. Open `index.html` in a browser.
2. Ensure internet access for the Tailwind CSS CDN.
3. Customize text, images, links, and styling as needed.

## Customization

- Replace the placeholder restaurant logo and branding text
- Update navigation links and section content
- Swap images in `img/` for your own photography or illustrations
- Edit Tailwind utility classes directly in `index.html`

## Notes

- This is a static template; form submission is not connected to a backend.
- The mobile menu opens and closes using simple inline JavaScript in `index.html`.
